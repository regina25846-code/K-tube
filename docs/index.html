⬇ 포터블 다운로드 (.exe)
시작 전에 — YouTube API 키 설정

🔒 API 키는 내 컴퓨터에만 저장됩니다.
입력한 API 키는 이 앱 외부로 전송되거나 공유되지 않습니다. 로컬 파일에만 저장되며, 오직 YouTube 영상 검색·조회에만 사용됩니다.

⚡ API 키를 여러 개 준비하는 이유
YouTube Data API는 하루 무료 쿼터가 10,000점으로 제한됩니다. 영상 검색·조회 등을 사용할 때마다 점수가 차감되며, 소진되면 당일 자정까지 검색이 중단됩니다.

프로젝트를 여러 개 만들어두면 쿼터 소진 시 키를 전환하며 끊김 없이 사용할 수 있습니다. K-Tube는 등록된 키가 여러 개이면 자동으로 순환 사용합니다.

STEP 1
Google Cloud Console 접속
console.cloud.google.com에 접속 후 Google 계정으로 로그인합니다.

처음 접속하면 서비스 약관 동의 화면이 나옵니다. 국가를 선택하고 약관에 동의 후 계속하면 됩니다.
※ 이메일 업데이트 수신은 체크 안 해도 됩니다.
STEP 2
새 프로젝트 만들기
상단 프로젝트 선택 → 새 프로젝트 클릭.
이름은 K-Tube_1, K-Tube_2 처럼 자유롭게 지정하세요.

💡 쿼터 소진 시 새 프로젝트로 전환하는 방법
하나의 프로젝트에서 API 키를 여러 개 만들어도 쿼터는 프로젝트 단위로 공유됩니다.
쿼터를 늘리려면 프로젝트 자체를 새로 만들어야 합니다.
K-Tube_1 쿼터 소진 → K-Tube_2 프로젝트 새로 만들기 → 거기서 API 키 발급 → 앱 설정에 추가
앱에 여러 키를 등록해두면 자동으로 순환하며 사용됩니다.
STEP 3
YouTube Data API v3 활성화
왼쪽 메뉴 → API 및 서비스 → 라이브러리 → YouTube Data API v3 검색 → 사용 설정.
STEP 4
API 키 발급
API 및 서비스 → 사용자 인증 정보 → 사용자 인증 정보 만들기 → API 키 클릭.
생성된 키를 복사해두세요.
STEP 5
K-Tube에 키 입력
앱 실행 후 우측 상단 설정에서 API 키를 붙여넣기하면 완료됩니다.
하루 쿼터(무료 10,000회)가 소진되면 다른 프로젝트의 키로 추가해서 사용하세요.
